== Description ==
 
WooCommerce integration for AmautaLMS. Sell your courses using WooCommerce products.
 
This plugin add a tab on course editing to add one or more products related to course.

Also works with WooCommerce Subscriptio and CodeCanyon Subscriptio.
 
More info: [Get Amauta](https://amautawp.com)

Guides and examples in spanish for entrepreneurs: [Visit Emprende360](https://emprende360.net)
 
== Installation ==
  
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
